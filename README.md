### 🐻 웅녀 게임
---

교내 게임 개발 동아리 NPC에서 lua를 사용하여 만든 게임이다. 


#### 💡프로젝트 소개
---

웅녀가 사람이 되는 과정을 담을 게임. 다양한 미니 게임을 통해 경험치를 얻고, 경험치에 따라 다양한 엔딩 루트를 경험할 수 있다!
